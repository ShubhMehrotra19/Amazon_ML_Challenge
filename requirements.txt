#    - Mac: brew install libjpeg
#
# 2. XGBoost/LightGBM installation fails:
#    - Try: pip install --upgrade pip setuptools wheel
#    - Then retry installation
#
# 3. Memory errors during execution:
#    - Reduce CONFIG['text_processing']['max_features'] to 1000
#    - Reduce CONFIG['feature_engineering']['image_sample_size'] to 1000
#    - Set QUICK_TEST = True in main execution
#
# 4. Image download failures:

#    - Increase timeout in _extract_single_image_features() if needed
#    - Consider running with use_real_image_features=False for faster execution
